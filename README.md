# Pianana

Piano hecho con Arduino + NodeJS y Bananas.

#####Requisitos
  - Arduino
  - NodeJS
  - Johnny-Five
  - Protoboard
  - Zumbador

La idea es hacer un piano con bananas y al tocar las bananas se 
reproduzca un tono por cada banana.

### Uso

```sh
$ git clone https://github.com/andyalonzo/Pianana.git
$ cd Pianana
$ node index.js
```
